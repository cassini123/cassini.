# 纪元 · 透明共生 Arduino 外壳

## 💡 一、项目概述

本项目基于HTAM课程，构想一个未来的可能，并结合 3D 打印与激光切割技术制作arduino外壳，不使用胶水，仅通过机械结构和螺丝进行连接。



## 🔮 二、我的构想的未来：透明共生纪元

“透明共生纪元” 的未来时代。在这个未来，社会对科技产品的要求不再仅仅是功能强大，更必须是 **可理解、可循环、可信任**。

| 理念维度 | 核心概念 | 在设计中的体现 |
| :--- | :--- | :--- |
| **生态共生** | **从自然来，回归自然去。** | 选择 **PLA** 作为主要材料，它源于植物，具备最终 **可堆肥降解** 的潜力，代表科技产品对环境的终极责任。 |
| **技术透明** | **破除技术的“黑箱”效应。** | 使用 **亚克力透明视窗**。用户可以清晰观察内部 Arduino 板的运行状态、LED 闪烁和线路连接，象征着 **开源精神、可审计性** 和对用户充分知情权的尊重。 |

## 🎨 三、设计理念与产品介绍

本设计演化出两个版本，以适应不同的功能需求和理念侧重。


### 版本一：生态之锚（全闭合 PLA 盒子）——主要用于存放

**特点：**
* **材料：** 纯 **PLA**。
* **结构：** 3D 打印上下两部分，通过 **卡扣结构** 闭合连接。
* **设计理念：** 强调 **PLA** 材质的 **坚固性、轻量化和防尘保护**。由于 PLA 优秀的刚性和易于成型的特性，它为内部电子元件提供了坚实的屏障，适合用于 **长期固定、无需频繁观察** 的数据采集或控制节点。此版本是最纯粹的“共生”版本，完全致力于环保和保护。

### 版本二：透明之眼（PLA 主体 + 亚克力视窗）——用于存放和观察arduino运行状态

**特点：**
* **材料：** PLA、亚克力、热熔螺母、M3 螺丝。
* **结构：** PLA 主体提供底层保护，**激光切割亚克力面板** 作为顶盖。顶盖通过 **热熔螺母** 植入 PLA 上盖，再通过螺丝实现 **可开合** 的连接，方便维护。
* **设计理念：** 在继承版本一的生态理念上，加入了 **“透明”** 的核心价值。亚克力视窗便于调试和教学，允许使用者 **直接观察** Arduino 的运行状态。**可开合** 的设计体现了产品的 **可维护性和模块化**，鼓励用户参与到硬件的升级和修改中。

## 🛠️ 四、技术特点与材料清单

### 1. 结构

* **无胶水连接：** 所有连接均依赖于 **卡扣、热熔螺母和螺丝**，确保了产品的易拆卸性、可维护性和高重复利用率。
* **可开合顶盖：** 可以随时打开，方便地对 Arduino 主板进行接线或升级操作。

### 2. 使用材料清单

| 类别 | 物品 | 规格/用途 |
| :--- | :--- | :--- |
| **核心主体** | PLA 3D 打印件 | 外壳的上下壳体、结构框架（绿色/白色，易于降解的生态色） |
| **透明组件** | 透明亚克力板 | 1mm 厚，用于激光切割可视窗口（版本二） |
| **连接硬件** | M3 热熔螺母 | 植入 PLA 顶盖，提供可靠的螺纹连接基座（版本二） |
| **连接硬件** | M3 沉头/平头螺丝 | 用于固定亚克力面板和连接上下壳体 |
| **电子元件** | Arduino UNO/Nano 等 | 存放于外壳内部的假想产品主控板 |
